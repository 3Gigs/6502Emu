# Building
* Clone this directory  
* Go to /build folder  
  ```cd {PATH_TO_CLONED_DIRECTORY}/6502Emu/build```
* Set srcPath directory (For Makefile)  
  ```export srcPath='{PATH_TO_SRC_DIRECTORY}'```
